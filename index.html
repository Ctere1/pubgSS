ℹ️ Introduction

This application helps PUBG (PlayerUnknown's Battlegrounds) players analyze historical match data to detect potential stream sniping behavior. Stream sniping occurs when a player watches a live broadcast to gain an unfair tactical advantage.

📊

JSON Export

Export the complete structured match dataset for external inspection or custom tooling.

📄

PDF Reports

Generate polished, printable analytical summaries for documentation or evidence submission.

⚖️

Player Comparison

Compare two distinct players' recurrence patterns, proximity timelines and match overlap rates.

Important: This application does not run in real time. It only displays historical match data from the official PUBG API.
