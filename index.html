ℹ️ Introduction

This application is designed to help players of PUBG (PlayerUnknown's Battlegrounds) detect potential stream snipers. Stream sniping is a form of cheating where players watch a live stream of another player to gain an unfair advantage in the game.

📊

JSON Export

Export all match data to a JSON file for detailed analysis

📄

PDF Reports

Generate comprehensive PDF reports of match data

⚖️

Player Comparison

Compare two different players' match data side by side

Important: This application does not run in real time; it only displays match history data retrieved from the PUBG API.
